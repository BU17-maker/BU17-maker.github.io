# 个人网站项目概述 (v2版本)

## 项目概览

这是一个专为高中生/OIer设计的极简风格个人技术博客，采用黑白灰配色方案，专注于实用性和移动端体验。网站体现了少年感的设计风格，内容围绕算法竞赛、编程学习和高考规划。

## 核心特色

### 极简设计
- **严格配色**：仅使用 #000、#fff、#111、#222、#555、#999 六种颜色
- **无装饰效果**：删除所有渐变、霓虹、阴影、透明度动画
- **1px实线边框**：简洁的边框设计
- **瞬时交互**：hover效果≤50ms，无放大旋转

### 少年感内容
- **个人简介**：高二男生，停更OIer，C++学习者
- **项目展示**：洛谷题解、GitHub博客等学习项目
- **学习规划**：高考前的技术学习路线图
- **实用工具**：算法竞赛相关工具收藏

### 移动端优先
- **响应式设计**：先移动端设计，逐步增强
- **流式布局**：CSS Grid和Flexbox自适应
- **触控优化**：48×48px最小触控区域
- **字体缩放**：clamp()函数实现平滑缩放

## 页面结构

### 1. 个人主页 (index.html)
**功能定位**：个人介绍和核心信息展示
- **Logo区域**："我的工具箱"中文标识
- **个人简介**：高二男生，OIer身份，技术学习方向
- **社交链接**：GitHub、洛谷等学习平台链接
- **功能介绍**：网站各板块功能说明
- **访问信息**：当前时间和IP地址显示

### 2. 工具收藏夹 (tools.html)
**功能定位**：算法竞赛相关工具和资源整理
- **分类管理**：OJ平台、可视化工具、文档资源、美化工具、图床服务
- **搜索过滤**：支持按名称、描述、分类搜索
- **一键复制**：Markdown格式复制功能
- **直达链接**：新标签页打开工具网站
- **工具统计**：收藏工具数量统计

### 3. 学习路线图 (roadmap.html)
**功能定位**：高考前的技术学习规划
- **时间轴布局**：垂直时间轴展示学习计划
- **阶段性目标**：每个时间段的具体学习任务
- **进度跟踪**：学习进度和完成状态
- **展开/收起**：详细内容可折叠显示
- **规划统计**：总计划数、完成进度统计

### 4. 项目展示 (projects.html)
**功能定位**：个人学习项目和技术实践
- **项目卡片**：简洁的项目信息展示
- **详情模态框**：点击查看项目详细信息
- **技术标签**：使用的技术栈标识
- **项目链接**：GitHub代码仓库链接
- **项目感想**：学习心得和技术总结

### 5. C++代码模板 (cpp.html)
**功能定位**：算法竞赛代码模板库
- **树形目录**：分类管理代码模板
- **代码高亮**：语法高亮显示
- **代码操作**：复制、下载功能
- **响应式布局**：移动端优化的左右分栏
- **模板示例**：快速排序、二分查找、图论算法等

### 6. 错题本 (mistakes.html)
**功能定位**：算法题目错误记录和复习管理
- **错题表格**：题目信息、错误原因、复习状态
- **状态管理**：未复习、复习中、已复习状态切换
- **搜索过滤**：按题目、原因、标签搜索
- **题解链接**：一键跳转到题解页面
- **复习统计**：错题数量、复习进度统计

## 技术架构

### 前端技术栈
- **HTML5**：语义化标签，表单验证
- **CSS3**：Flexbox、Grid布局，CSS变量
- **原生JavaScript**：ES6+语法，模块化开发

### 核心库集成
- **Font Awesome 6.5**：图标系统
- **AOS.js**：滚动动画（简化版）
- **Prism.js**：代码语法高亮

### 设计系统
- **CSS变量**：统一主题管理
- **组件化**：可复用的UI组件
- **响应式**：移动端优先的设计

## 设计规范

### 颜色使用
```css
:root {
    --bg-color: #000;        /* 背景色 */
    --text-color: #fff;      /* 主要文字 */
    --card-bg: #111;         /* 卡片背景 */
    --border-color: #222;    /* 边框颜色 */
    --text-muted: #999;      /* 次级文字 */
}
```

### 字体系统
```css
/* 基础字体 */
font-size: clamp(1rem, 0.95rem + 0.25vw, 1.125rem);
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

/* 代码字体 */
font-family: 'Consolas', 'Monaco', 'Courier New', monospace;
```

### 间距系统
```css
:root {
    --spacing-xs: 4px;
    --spacing-sm: 8px;
    --spacing-md: 16px;
    --spacing-lg: 24px;
    --spacing-xl: 32px;
}
```

## 交互设计

### 导航交互
- **中文菜单**：首页、工具、规划、项目、模板、错题本
- **汉堡菜单**：≤600px时折叠为"≡"图标
- **背景切换**：hover时#111→#222，≤50ms过渡
- **垂直展开**：移动端点击垂直展开菜单

### 卡片交互
- **背景切换**：hover时#111→#222瞬时切换
- **无放大效果**：保持原始尺寸
- **焦点指示**：2px白色轮廓线
- **键盘导航**：支持Tab键导航

### 模态框交互
- **点击打开**：卡片点击打开详情
- **多种关闭**：关闭按钮、背景点击、ESC键
- **焦点管理**：打开时聚焦，关闭时返回
- **滚动锁定**：打开时禁止背景滚动

## 响应式设计

### 断点系统
- **小屏幕**：≤480px (移动端)
- **中屏幕**：481px - 600px (大手机)
- **大屏幕**：>600px (平板及以上)

### 布局适配
- **≤480px**：单列布局，卡片堆叠
- **481px-600px**：单列布局，更大间距
- **>600px**：多列布局，CSS Grid自适应

### 导航适配
- **≤600px**：汉堡菜单，垂直展开
- **>600px**：水平导航栏

## 性能优化

### 资源优化
- **内联CSS**：关键样式内嵌减少HTTP请求
- **CDN资源**：第三方库使用CDN加速
- **图片优化**：背景图压缩处理

### 代码优化
- **事件委托**：减少事件监听器数量
- **防抖搜索**：搜索输入防抖处理
- **模块化JS**：按功能拆分JavaScript代码

### 移动端优化
- **触控优化**：48×48px最小触控区域
- **滚动优化**：平滑滚动，惯性滚动
- **内存管理**：及时清理事件监听器

## 无障碍设计

### 颜色对比度
- **主要文字**：#fff on #000 → 21:1 (AAA级)
- **次级文字**：#999 on #000 → 7.4:1 (AAA级)
- **UI元素**：#111 on #000 → 1.5:1 (满足要求)

### 键盘导航
- **Tab顺序**：逻辑清晰的焦点顺序
- **焦点指示**：2px白色实线轮廓
- **快捷键**：ESC关闭模态框，Enter确认操作

### 屏幕阅读器
- **语义化HTML**：正确使用HTML5标签
- **ARIA标签**：重要元素添加aria-label
- **标题层级**：h1-h6正确使用

## 开发指南

4. 在`main.js`中添加页面初始化函数

1. 遵循黑白灰配色方案
2. 使用CSS变量统一管理
3. 保持hover效果一致性
4. 测试移动端适配
