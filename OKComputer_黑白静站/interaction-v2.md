# 个人网站交互设计 (v2版本)

## 核心交互功能

### 1. 导航栏交互
- **中文导航菜单**：首页、工具、规划、项目、模板、错题本
- **Logo文字**："我的工具箱"
- **汉堡菜单**：≤600px时自动折叠为"≡"图标，点击垂直展开
- **背景色切换**：hover时背景色从#111切换到#222，无放大效果
- **过渡时间**：≤50ms，支持prefers-reduced-motion时直接跳变

### 2. 主页交互
- **社交链接卡片**：hover时背景色#111→#222，无3D翻转
- **头像和文字**：保持静态，无动画效果
- **功能介绍**：简洁的文字描述，无复杂动画

### 3. 工具收藏页交互
- **搜索功能**：支持按名称、描述、标签搜索
- **分类过滤**：全部、OJ平台、可视化工具、文档资源、美化工具、图床服务
- **工具卡片**：hover背景色切换，点击"直达"按钮新标签页打开
- **复制功能**：点击复制图标复制Markdown格式
- **响应式布局**：≤480px时单列显示，>480px时CSS Grid自适应

### 4. 路线图页交互
- **时间轴节点**：点击可展开/收起详细内容
- **状态显示**：进行中、计划中、已完成
- **进度指示**：简单的进度条显示
- **垂直布局**：移动端优化的单列时间轴

### 5. 项目展示页交互
- **项目卡片**：hover背景色#111→#222
- **模态框**：点击卡片打开详情，包含项目描述、技术栈、感想
- **关闭方式**：点击关闭按钮或背景关闭
- **键盘支持**：ESC键关闭模态框

### 6. C++模板页交互
- **树形目录**：点击文件夹展开/收起，点击文件加载代码
- **代码显示**：语法高亮，支持复制和下载
- **响应式设计**：移动端优化的左右布局
- **文件操作**：支持代码复制、文件下载

### 7. 错题本页交互
- **状态切换**：点击状态标签循环切换（未复习→复习中→已复习）
- **搜索过滤**：支持按题目名称、错误原因、标签搜索
- **状态过滤**：全部、未复习、复习中、已复习
- **表格操作**：去刷题按钮跳转到题解链接

## 移动端适配

### 响应式断点
- **≤480px**：单列布局，卡片堆叠显示
- **>480px**：CSS Grid布局，auto-fit自适应
- **≤600px**：导航栏折叠为汉堡菜单
- **>600px**：导航栏水平展开

### 触控优化
- **最小触控区域**：48×48px
- **按钮间距**：足够大的间隔防止误触
- **滚动优化**：表格在小屏设备支持横向滚动

### 字体缩放
- **基准字体**：16px
- **响应式字体**：clamp(1rem, 0.95rem + 0.25vw, 1.125rem)
- **标题字体**：根据层级递减

## 无障碍设计

### 动画控制
- **prefers-reduced-motion**：用户关闭动画时直接跳变
- **过渡时间**：≤50ms，避免眩晕感
- **焦点样式**：清晰的焦点指示器

### 语义化结构
- **HTML5语义标签**：header、nav、main、section、footer
- **ARIA属性**：适当的aria-label和role属性
- **键盘导航**：支持Tab键导航所有交互元素

## 性能优化

### 资源加载
- **CDN资源**：Font Awesome、第三方库使用CDN
- **图片优化**：背景图压缩，响应式图片
- **代码分离**：CSS和JS内嵌减少HTTP请求

### 交互优化
- **防抖搜索**：搜索输入防抖处理
- **懒加载**：图片懒加载（如需要）
- **事件委托**：合理使用事件委托减少内存占用

## 用户体验

### 视觉层次
- **黑白灰配色**：#000背景，#fff文字，#111/#222/#555/#999灰色层级
- **1px实线边框**：简洁的边框设计
- **无阴影效果**：保持扁平化设计风格

### 内容可读性
- **行高1.7**：舒适的阅读体验
- **字符限制**：每行最多65字符
- **对比度**：确保4.5:1的对比度比例

### 交互反馈
- **即时反馈**：hover效果≤50ms响应
- **状态指示**：清晰的加载、成功、错误状态
- **操作确认**：重要操作提供确认机制

## 技术实现

### 前端技术
- **HTML5**：语义化标签，表单验证
- **CSS3**：Flexbox、Grid布局，CSS变量
- **原生JavaScript**：ES6+语法，模块化开发

### 兼容性
- **现代浏览器**：Chrome、Firefox、Safari、Edge
- **移动浏览器**：iOS Safari、Chrome Mobile
- **渐进增强**：基础功能在所有浏览器可用

### 数据管理
- **LocalStorage**：用户偏好设置
- **SessionStorage**：临时数据存储
- **集中配置**：userData.js统一管理

## 开发指南

### 添加新页面
1. 复制index.html作为模板
2. 修改<main>区域内容
3. 在_navbar.html中添加导航链接
4. 在main.js中添加页面初始化函数

### 修改样式
1. 遵循黑白灰配色方案
2. 使用CSS变量统一主题
3. 保持hover效果一致性
4. 确保移动端适配

### 添加交互
1. 使用事件委托优化性能
2. 添加键盘导航支持
3. 考虑prefers-reduced-motion
4. 提供清晰的视觉反馈
